# Requirements

- node 18.x
- npm/pnpm

# Stack Used

- node
- typescript
- pnpm
- jest

# How To Run code

For installing all the dependencies

1)For pnpm package manger

```
pnpm i
```
2)For npm package manger
```
npm install
```

For compiling code

```
npm run tsc
```

For running test cases

```
npm run test
```
