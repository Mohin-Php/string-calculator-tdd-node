# Requirements

- node 18.x
- npm/pnpm

# Stack Used

- node
- typescript
- pnpm
- jest

# How To Run code

For installing all the dependencies

- For pnpm package manger

```
pnpm i
```
- For npm package manger
```
npm install
```

- For compiling code

```
npm run tsc
```

- For running test cases

```
npm run test
```
